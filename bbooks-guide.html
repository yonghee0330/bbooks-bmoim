  • 오픈 인스타 게시 (개인 계정)

    • 비북스 본계에서도 함께 업로드 예정

💳
계산 (토스 포스)
  • 바탕화면 토스 포스 실행
  • '판매' 탭 → 바코드 스캔 or 제목 검색
  • 결제 방식 선택 → 카드 / 계좌이체 / 현금

    • 현금은 가급적 계좌이체 권장
    불가피한 경우 작업실 서랍 현금시제 사용 + 봉투에 내역 기입

  • 전자영수증 원하면 핸드폰 번호로 발행
  • 회원 등록 제안 → "간단히 등록 도와드릴까요?"

    • 포스 고객정보 → 연락처 항목을 이름(지역)으로 수정
    메모란에 취향 키워드 3개 기입

📌 별도 바코드 없는 도서 = 비북스 자체 상품
📌 바코드 붙은 도서·굿즈 = 0.1평 서점주 상품
🎁 연락처 등록 시 1% 적립 프로모션 세팅 중 (비북스 부담)
